Techlog - 閲覧履歴活用アプリ

Python Flask JavaScript Firebse chrome拡張機能

大学の卒業制作として作成した作品。
現代の情報化社会において、エンジニアは常に最新の技術情報をインプットし続けることが求められる。
しかし、膨大な情報量に対し、単に閲覧しただけの知識は定着せず、技術へと消化されないまま忘れられてしまうという課題がある。
本研究ではこの課題に対し、情報の量ではなく質と定着に焦点を当てたwebサービス「TechLog」を開発した。
本システムはブラウザの閲覧履歴からのAIによる情報の自動収集と要約、エビングハウスの忘却曲線に基づいた定期的な振り返り機能を統合したものである。
これによりユーザーは収集の手間をかけずに内面的な気づきや学びの記録に集中することが可能となる。
本制作は自身が直面する学習の課題から技術習得のプロセスをシステム化したものである。
